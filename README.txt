OA Store Theme

Installation:
1. WordPress Admin -> Appearance -> Themes -> Add New -> Upload Theme.
2. Upload oa-store-theme-installable.zip.
3. Activate "OA Store Theme".
4. If checkout page does not appear immediately, switch to another theme then reactivate OA Store Theme once.

Included templates:
- front-page.php
- page-checkout.php (Template Name: Checkout Page)
